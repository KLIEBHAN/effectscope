# useEffect Lab

Interaktive Lern-App, die `useEffect` in React anhand von Live-Demos, Code und einem Event-Log erklärt.

## Start

```bash
npm install
npm run dev
```

## Lektionen

0. Was ist useEffect?
1. Nur beim Mount (`[]`)
2. Bei jedem Render (ohne Array)
3. Dependencies
4. Cleanup
5. Fetch & Race Conditions
6. Endlosschleifen-Fallstrick
